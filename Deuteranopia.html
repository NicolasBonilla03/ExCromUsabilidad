Deuteranopia

En este tipo de daltonismo las personas no pueden distinguir el color verde y esto dificulta la distinción entre los colores verde y rojo.

Pauta


-Legibilidad del texto: Al combinar el color del fondo, el color del texto y el tamaño del texto se puede lograr que el texto sea más fácil de leer.

Tip


Para este tipo de daltonismo se debe evitar el uso de colores como el verde y el rojo, mejor usar unos colores con buen contraste que ellos pueden distinguir como el azul y el naranja.
