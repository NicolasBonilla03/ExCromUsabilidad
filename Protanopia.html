Protanopia

En este tipo de daltonismo las personas no pueden distinguir el color rojo y esto dificulta la distinción de los colores verde y rojo.

Pauta


Información con color: Evitar que la información de lo que se ve se transmita mediante el uso del color, por ejemplo para unos botones de correcto e incorrecto no solo usan el rojo y el verde sino también usar un icono que indique cual es cada uno.

Tip


Para este tipo de daltonismo se debe evitar el uso de colores como el verde y el rojo, mejor usar unos colores con buen contraste que ellos pueden distinguir como el azul y el amarillo.
